# AM-ARM200

AlohaMini2 uses AM-ARM200 as its follower and leader arm platform.

The arm CAD, assembly guide, firmware notes, and updates are maintained in the upstream repository:

**[liyiteng/AM-ARM](https://github.com/liyiteng/AM-ARM)**

Use the AM-ARM200 documentation there when building or modifying the arms for AlohaMini2.
